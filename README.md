# Smart City Dashboard

## Overview
This project is an interactive Power BI dashboard designed to analyze smart city data including crime analysis, air quality, traffic insights, and population statistics.

## Features
- Executive Overview Dashboard
- Crime Analysis
- Air Quality Monitoring
- Traffic Analysis
- Interactive Visualizations
- Dynamic Filters
- India Map Integration using JSON

## Tools & Technologies
- Power BI
- Power Query
- DAX
- JSON Map
- CSV/Excel Datasets

## Project Structure
- PowerBI_Dashboard → Main Power BI file
- Dataset → Source datasets
- Assets → Logos, wallpapers, JSON files
- Screenshots → Dashboard preview images
